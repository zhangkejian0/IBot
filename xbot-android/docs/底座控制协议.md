# XBot 底座控制协议

> 版本：1.0.0 · 最后更新：2026-06-22
> 适用设备：XBot 桌面 AI 陪伴机器人底座（2/3 自由度旋转）

---

## 一、概述

### 1.1 协议目的

本文档定义 XBot 应用与旋转底座之间的**数据交互格式**。协议采用 JSON 格式描述指令和状态，不限定具体传输方式。

App 通过适配器层将 JSON 指令转换为具体传输协议（BLE / WiFi / Serial / 自定义协议），实现与不同底座设备的兼容。

### 1.2 应用场景

| 场景 | 说明 | 输入源 |
|------|------|--------|
| 人脸跟随 | 底座朝向用户面部 | 人脸检测坐标 |
| 手势指向 | 底座朝向手部位置 | 手势检测坐标 |
| 语音定向 | 底座朝向声源方向 | 声源定位 |
| 手动控制 | 用户通过 UI 控制角度 | 用户输入 |
| 自动巡航 | 底座按预设轨迹运动 | 预设程序 |

## 二、坐标系统

### 2.1 旋转轴定义

支持 **2 自由度（Yaw + Pitch）** 基础配置，预留 **3 自由度（+ Roll）** 扩展。

```
                    Pitch (Tilt)
                        ↑
                        │
           ┌────────────┼────────────┐
           │            │            │
  Yaw ─────┼────────────┼────────────┼─────▶
  (Pan)    │            │            │
           └────────────┼────────────┘
                        │
                        ▼
```

| 轴 | 英文 | 旋转方向 | 说明 |
|----|------|----------|------|
| Yaw | Pan | 绕垂直轴 | 控制左右朝向 |
| Pitch | Tilt | 绕水平轴 | 控制上下朝向 |
| Roll | Roll | 绕前后轴 | 控制歪头（3 DOF 扩展） |

### 2.2 角度范围

| 轴 | 范围 | 原点 | 精度 | 说明 |
|----|------|------|------|------|
| Yaw | -90° ~ +90° | 0° | 0.1° | 负=左，正=右 |
| Pitch | -30° ~ +30° | 0° | 0.1° | 负=上，正=下 |
| Roll | -45° ~ +45° | 0° | 0.1° | 负=左歪，正=右歪（3 DOF） |

> 原点位置可通过 `calibrate` 命令动态设置。

### 2.3 归一化坐标

协议采用归一化坐标（-1..1）作为输入，与具体感知引擎解耦。

| 坐标 | 范围 | 正方向 | 说明 |
|------|------|--------|------|
| targetX | -1.0 ~ +1.0 | 右 | 水平目标位置 |
| targetY | -1.0 ~ +1.0 | 下 | 垂直目标位置 |
| targetZ | -1.0 ~ +1.0 | 顺时针 | 横滚目标（3 DOF） |

**映射公式**：

```
Yaw   = clamp(targetX, -1, 1) × yawRange
Pitch = clamp(targetY, -1, 1) × pitchRange
Roll  = clamp(targetZ, -1, 1) × rollRange   (仅 3 DOF)
```

## 三、指令格式

所有指令和响应均为 JSON 对象，单行传输。

### 3.1 指令列表

| 指令 | 方向 | 说明 |
|------|------|------|
| `move` | App → Base | 移动到目标角度 |
| `move_rel` | App → Base | 相对移动 |
| `home` | App → Base | 回到原点 |
| `stop` | App → Base | 立即停止 |
| `set_mode` | App → Base | 设置工作模式 |
| `set_speed` | App → Base | 设置运动速度 |
| `calibrate` | App → Base | 校准原点 |
| `get_status` | App → Base | 查询状态 |
| `get_version` | App → Base | 查询固件版本 |
| `heartbeat` | App ↔ Base | 心跳保活 |
| `ack` | Base → App | 命令确认 |
| `status` | Base → App | 状态上报 |
| `angle` | Base → App | 角度上报 |
| `version` | Base → App | 固件版本 |
| `error` | Base → App | 错误报告 |

### 3.2 指令详情

#### move - 移动到目标角度

**请求**：
```json
{
  "cmd": "move",
  "yaw": 45.0,
  "pitch": -15.0,
  "roll": 10.0,
  "id": 1
}
```

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| yaw | float | 是 | 目标 Yaw 角度（度） |
| pitch | float | 是 | 目标 Pitch 角度（度） |
| roll | float | 否 | 目标 Roll 角度（3 DOF） |
| id | int | 否 | 请求 ID |

**响应**：
```json
{"cmd": "ack", "ref": 1, "ok": true}
```

---

#### move_rel - 相对移动

**请求**：
```json
{
  "cmd": "move_rel",
  "dyaw": 10.0,
  "dpitch": -5.0,
  "droll": 5.0,
  "id": 2
}
```

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| dyaw | float | 是 | Yaw 增量（度） |
| dpitch | float | 是 | Pitch 增量（度） |
| droll | float | 否 | Roll 增量（3 DOF） |

---

#### home - 回到原点

**请求**：
```json
{"cmd": "home", "id": 3}
```

---

#### stop - 立即停止

**请求**：
```json
{"cmd": "stop", "id": 4}
```

---

#### set_mode - 设置工作模式

**请求**：
```json
{"cmd": "set_mode", "mode": "track", "id": 5}
```

| 字段 | 类型 | 可选值 | 说明 |
|------|------|--------|------|
| mode | string | `idle` `track` `manual` `sleep` `demo` | 工作模式 |

**模式说明**：

| 模式 | 说明 |
|------|------|
| idle | 空闲，电机释放 |
| track | 追踪模式，接收目标角度自动跟随 |
| manual | 手动模式，直接控制 |
| sleep | 休眠，锁定当前位置 |
| demo | 演示模式，自动巡航 |

---

#### set_speed - 设置运动速度

**请求**：
```json
{
  "cmd": "set_speed",
  "yaw_speed": 80,
  "pitch_speed": 60,
  "roll_speed": 50,
  "id": 6
}
```

| 字段 | 类型 | 范围 | 说明 |
|------|------|------|------|
| yaw_speed | int | 0~100 | Yaw 轴速度百分比 |
| pitch_speed | int | 0~100 | Pitch 轴速度百分比 |
| roll_speed | int | 0~100 | Roll 轴速度（3 DOF） |

---

#### calibrate - 校准

**请求**：
```json
{"cmd": "calibrate", "type": "set_home", "id": 7}
```

| 字段 | 类型 | 可选值 | 说明 |
|------|------|--------|------|
| type | string | `set_home` `full` | 校准类型 |

- `set_home`：设置当前位置为原点
- `full`：完整校准流程（限位扫描）

---

#### get_status - 查询状态

**请求**：
```json
{"cmd": "get_status", "id": 8}
```

**响应**：见 `status` 指令

---

#### get_version - 查询固件版本

**请求**：
```json
{"cmd": "get_version", "id": 9}
```

**响应**：见 `version` 指令

---

#### heartbeat - 心跳

```json
{"cmd": "heartbeat", "ts": 1719052800}
```

收到后原样回复，用于连接保活。

---

### 3.3 响应详情

#### ack - 命令确认

```json
{"cmd": "ack", "ref": 1, "ok": true}
```

| 字段 | 类型 | 说明 |
|------|------|------|
| ref | int | 对应请求的 id |
| ok | bool | 是否成功 |

---

#### status - 状态上报

```json
{
  "cmd": "status",
  "mode": "track",
  "yaw": 44.8,
  "pitch": -14.9,
  "roll": 10.0,
  "dof": 3,
  "moving": true,
  "calibrated": true,
  "error": null,
  "id": 8
}
```

| 字段 | 类型 | 说明 |
|------|------|------|
| mode | string | 当前工作模式 |
| yaw | float | 当前 Yaw 角度 |
| pitch | float | 当前 Pitch 角度 |
| roll | float | 当前 Roll 角度（3 DOF） |
| dof | int | 设备自由度（2 或 3） |
| moving | bool | 是否正在运动 |
| calibrated | bool | 是否已校准 |
| error | string | 错误信息（无错误为 null） |

---

#### angle - 角度上报

```json
{"cmd": "angle", "yaw": 44.8, "pitch": -14.9, "roll": 10.0, "ts": 1719052800}
```

---

#### version - 固件版本

```json
{
  "cmd": "version",
  "major": 1,
  "minor": 0,
  "patch": 0,
  "build": 1,
  "dof": 3,
  "hw": "XBot-Base-3DOF"
}
```

| 字段 | 类型 | 说明 |
|------|------|------|
| major ~ build | int | 语义化版本号 |
| dof | int | 支持的自由度（2 或 3） |
| hw | string | 硬件型号 |

---

#### error - 错误报告

```json
{"cmd": "error", "code": "limit_hit", "msg": "Yaw axis limit triggered", "ref": 1}
```

| 字段 | 类型 | 说明 |
|------|------|------|
| code | string | 错误码 |
| msg | string | 错误描述 |
| ref | int | 关联的请求 id |

**错误码**：

| code | 说明 |
|------|------|
| unknown_cmd | 未知命令 |
| invalid_param | 参数错误 |
| limit_hit | 限位触发 |
| overheat | 电机过热 |
| stall | 电机堵转 |
| timeout | 超时 |
| calibrate_fail | 校准失败 |

## 四、工作模式

### 4.1 状态机

```
                    ┌─────────────────────────────────────┐
                    │                                     │
                    ▼                                     │
┌──────┐    move    ┌──────┐    sleep    ┌──────┐        │
│ IDLE │◀──────────▶│TRACK │───────────▶│SLEEP │        │
└──────┘            └──────┘            └──────┘        │
   │                   │                   │             │
   │ manual            │ manual            │ manual      │
   ▼                   ▼                   ▼             │
┌──────┐            ┌──────┐                           │
│MANUAL│◀──────────▶│DEMO  │───────────────────────────┘
└──────┘   demo     └──────┘   home/stop
```

### 4.2 模式切换

| 当前模式 | 可切换到 | 触发指令 |
|----------|----------|----------|
| IDLE | TRACK, MANUAL, SLEEP, DEMO | set_mode, move |
| TRACK | IDLE, MANUAL, SLEEP, DEMO | set_mode, stop |
| MANUAL | IDLE, TRACK, SLEEP, DEMO | set_mode |
| SLEEP | IDLE, TRACK, MANUAL, DEMO | set_mode, move |
| DEMO | IDLE, TRACK, MANUAL, SLEEP | set_mode, home, stop |

## 五、控制参数建议

### 5.1 TRACK 模式

| 参数 | 建议值 | 说明 |
|------|--------|------|
| 控制周期 | 200ms | 5Hz 更新频率 |
| 平滑系数 | 0.6 | EMA 滤波 |
| 死区阈值 | ±0.5° | 小于此值不移动 |
| 最大单帧增量 | ±5°/帧 | 防止大幅跳变 |

### 5.2 安全保护

| 机制 | 说明 |
|------|------|
| 角度限幅 | 发送角度限制在设备范围内 |
| 速度限制 | 单帧最大增量限制 |
| 紧急停止 | stop 命令立即停止 |
| 过热保护 | 设备报告 overheat 时停止 |
| 限位保护 | 设备报告 limit_hit 时停止 |
| 堵转保护 | 设备报告 stall 时停止 |
