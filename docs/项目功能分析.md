## 1. 项目概述

**XBot（应用名「狗蛋」）** 是一个基于 Flutter 的**端侧多模态感知 + 语音对话**应用，定位为可摆在桌面上的「虚拟宠物 / 桌面伴侣」。

核心能力分两条线：

1. **端侧视觉感知**（离线、实时）：通过前置摄像头实时识别**人脸表情**、**手势**、**身份（认识谁）** 与**注视方向**。
2. **语音对话**（本地唤醒 + 云端对话）：本地离线唤醒词检测，唤醒后采集整段语音上传云端后端，完成「语音识别 → 大模型对话 → 语音合成」并播报。

两条线汇聚到一个**虚拟宠物形象**（WebView 内的网页桌面），把感知结果（表情/注视/语音状态）实时驱动到宠物的表情与动作上。

- 技术栈：Flutter（Dart SDK ^3.11.1），目标平台 Android / iOS
- 运行形态：强制横屏、全屏沉浸式、屏幕常亮（见 [main.dart](../lib/main.dart)）
- 状态管理：`ChangeNotifier` + `InheritedNotifier`（[AppScope](../lib/core/app_scope.dart)），无第三方状态库

## 2. 整体架构

```
main.dart  →  XBotApp(app.dart)  →  AppController(core/app_controller.dart)  ← 全局核心
                                          │
        ┌─────────────────┬──────────────┼───────────────┬──────────────────┐
   视觉感知引擎        身份识别        语音助手          虚拟桌面服务         持久化
        │                 │              │                │                  │
  FaceEngine        FaceRecognition  VoiceAssistant   StaticServer      PersonRepository
  HandEngine        Service          (编排器)         (本地HTTP托管)     各种 ConfigStore
  MlKitFaceEngine   PersonRepository      │
                                     ┌────┴──────────────────────┐
                                AudioCapture / WakeWord / Speech / Chat / PophieClient
```

[AppController](../lib/core/app_controller.dart) 是中枢：负责启动加载流程、相机取流，并把每一帧分发给各识别引擎，再把结果汇总成 `DetectionResult` 通知 UI 与虚拟宠物。

### 启动加载流程（`AppController.initialize`）
权限 → 相机 → 各模型 → 数据，逐步上报进度到 [LoadingScreen](../lib/ui/loading_screen.dart)：
1. 请求摄像头权限（拒绝则进入 `permissionDenied`）
2. 初始化相机（优先前置，锁定横屏取流）
3. 请求麦克风权限（失败不阻断，仅禁用语音）并后台预热语音助手
4. 加载人脸表情模型（MediaPipe）
5. 加载多脸检测（ML Kit）
6. 加载手势模型
7. 加载身份识别模型（MobileFaceNet）
8. 读取已录入人物 / 桌面配置 / LLM 配置 / Pophie 后端配置
9. 启动相机帧流，缓冲 2 秒后进入 `ready`

## 3. 功能模块详解

### 3.1 人脸表情识别
- 引擎：[FaceEngine](../lib/services/face_engine.dart)，封装 `kwon_mediapipe_landmarker`（MediaPipe Face Landmarker）。
- 输出：**478 个人脸关键点** + **52 个 ARKit 兼容 blendshapes**。
- 分类：[ExpressionClassifier](../lib/services/expression_classifier.dart) 用**规则式**（非训练模型）把 blendshapes 组合成 **7 种表情**：中性 / 高兴 / 伤心 / 惊讶 / 愤怒 / 厌恶 / 恐惧（见 [Expression](../lib/models/expression.dart)，Ekman 经典 6 种 + 中性）。

### 3.2 手势识别
- 引擎：[HandEngine](../lib/services/hand_engine.dart)，封装 `hand_detection`（MediaPipe 手部 + 手势，离线 TFLite）。
- 输出：每只手 **21 个关键点** + 左右手 + **7 种手势之一**（见 [hand_gesture.dart](../lib/models/hand_gesture.dart)）：握拳 / 张开手掌 / 食指向上 / 拇指向下 / 点赞 / 胜利 / 我爱你。
- 最多检测 2 只手。

### 3.3 身份识别（「认识我」）
- 多脸检测：[MlKitFaceEngine](../lib/services/mlkit_face_engine.dart)（Google ML Kit）返回每张脸的归一化包围盒，弥补 MediaPipe 单脸局限。
- 特征比对：[FaceRecognitionService](../lib/services/face_recognition_service.dart) 用 **MobileFaceNet（TFLite）** 把人脸裁剪图转成定长 embedding，按**余弦相似度**与已录入人物比对得出身份。模型缺失时该功能自动停用，不影响其它识别。
- **多人脸防串脸**：`AppController` 内用「位置身份槽（`_IdentitySlot`）」+ 贪心最近邻匹配，在归一化坐标空间稳定追踪每张脸，配 TTL（3s）让节流帧也能续上身份。
- 人物数据：[Person](../lib/models/person.dart) 含姓名 + 与主人的**家庭关系**（主人/配偶/父母/子女/兄弟姐妹/祖辈/朋友/其他）+ 头像 + embedding，由 [PersonRepository](../lib/services/person_repository.dart) 以 JSON + 头像图片本地持久化。
- 录入页：[FaceRegistrationScreen](../lib/ui/settings/face_registration_screen.dart) 为 Face-ID 风格横屏布局，自动采集 5 帧样本；已录入列表见 [friend_list_sheet.dart](../lib/ui/settings/friend_list_sheet.dart)。

### 3.4 注视方向
- [GazeZoneDetector](../lib/face/gaze_zone_detector.dart)：把屏幕分为 3×3 共 9 区，按人脸中心判断注视区域，带死区防边界抖动。
- [GazeSmoother](../lib/face/gaze_smoother.dart)：对连续注视值（-1..1）做指数平滑（EMA）。
- 用途：驱动虚拟宠物「看向」用户所在方向。

### 3.5 语音助手
编排器 [VoiceAssistant](../lib/services/voice/voice_assistant.dart) 是一个有限状态机（[VoiceState](../lib/services/voice/voice_state.dart)）：

```
idle → waking → listening → thinking → speaking → idle
（唤醒/双击触发）（采集语音）  （等LLM）   （TTS播报）
```

子服务：
- [AudioCaptureService](../lib/services/voice/audio_capture_service.dart)：麦克风 PCM 取流（16kHz/单声道/16-bit），并输出实时音量（驱动宠物嘴部张合）。
- [WakeWordService](../lib/services/voice/wake_word_service.dart)：**本地离线唤醒**，基于 `sherpa-onnx` KWS（wenetspeech 拼音建模）。唤醒词（默认「你好」）经 `lpinyin` 转带声调拼音 token，**支持运行时改词、无需重训模型**。
- [PophieClient](../lib/services/voice/pophie_client.dart)：核心对话通道。整段 WAV 上传 Pophie 后端 `/api/chat`，**一次完成 STT + LLM + TTS**，返回文本与 TTS 音频。携带端侧**多模态感知上下文**（表情 / 身份 / 手势）实现个性化回应；后端回传 `session_id` 持久化以延续会话记忆。详见 [API对接文档.md](API对接文档.md)。
- [ChatService](../lib/services/voice/chat_service.dart)：备选的 OpenAI 兼容 LLM 直连（默认 DeepSeek，可换通义/智谱/Kimi/自部署 vLLM），流式 chat completions。
- [SpeechService](../lib/services/voice/speech_service.dart)：ASR/TTS 抽象层（阿里云预留，当前为骨架）。

### 3.6 虚拟宠物 / 桌面
- 非调试模式下，主界面显示一个 **WebView 虚拟宠物**，由本地 HTTP 服务 [StaticServer](../lib/services/static_server.dart) 托管打包在 assets 中的网页（WebView 无法直接读 assets 相对路径，故先复制到临时目录再用 HTTP 提供）。
- 两套可切换桌面（[desktop_config.dart](../lib/core/desktop_config.dart)）：
  - **默认**：SVG 表情桌面（React/Vite 构建产物，环境光风格）
  - **魔女**：Live2D `monv` 模型
- 联动：[camera_screen.dart](../lib/ui/camera_screen.dart) 把检测结果（表情→[FaceState](../lib/face/emotion_mapper.dart)、注视、语音状态）以 `runJavaScript` 推送给网页，约 30fps（带节流）。

### 3.7 调试与可视化
- **调试模式**（设置页开关）：关闭时显示虚拟宠物，开启时显示摄像头识别画面。
- [DetectionOverlayPainter](../lib/ui/overlay_painter.dart) 叠加绘制：人脸关键点网格 / 人脸框 / 手部骨架 / 关键点序号 / 表情标签 / 手势标签 / 身份标签，均可在设置页单独开关。
- 性能优化：帧检测按 ~66ms（~15fps）节流，身份识别按 1.2s 节流，跳过帧复用上一帧结果避免闪烁。

### 3.8 设置页（[SettingsScreen](../lib/ui/settings/settings_screen.dart)）
iOS 风格分组设置：虚拟桌面选择 · 显示/调试模式 · 身份识别（认识我录入 / 我的朋友 / 开关）· 语音助手（总开关 / 唤醒开关 / 唤醒词 / 语音播报）· 识别功能开关（表情 / 手势）· 调试显示开关 · 关于（版本）。
